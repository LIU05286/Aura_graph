# Aura Graph · 记忆星图

Aura Graph 是一个纯前端、本地优先的 3D 知识图谱应用。它把你的想法、笔记、人物、项目、书籍、课程、事件和概念组织成一片可探索、可筛选、可编辑的星空宇宙。

当前新版本重点是：**多星系管理、AI 辅助、语义搜索、时间筛选、本地持久化**。

## 主要特性

- **3D 星空图谱**：基于 `React Three Fiber` + `three.js` 渲染节点、连线、星空背景和发光效果。
- **多星系管理**：支持创建、切换、重命名和删除星系；每个星系独立保存自己的图谱数据。
- **快速记录**：顶部快速输入一行内容即可创建一个新想法节点，并自动飞向该节点。
- **节点编辑**：支持创建、编辑、删除节点，维护标题、内容、类型、标签、重要度和时间信息。
- **关系管理**：在详情面板中为节点添加或删除关系，关系类型包含相关、因果、支持、矛盾、来源、相似、延展等。
- **搜索系统**：同时提供文本搜索和语义搜索；语义搜索通过 embedding 计算相似度排序结果。
- **筛选与聚焦**：可按节点类型、标签和时间范围筛选图谱，并支持一键聚焦到目标节点。
- **AI 辅助**：支持 AI 推荐标签和关系建议，用户确认后才会真正写入图谱。
- **导入 / 导出**：支持将当前图谱导出为 JSON，也可以从 JSON 导入替换当前数据。
- **示例重置**：一键恢复到内置示例图谱，方便演示和重新开始。
- **本地持久化**：使用 `Dexie + IndexedDB` 自动保存，刷新页面后数据仍然保留。

## AI 与接口说明

本项目里有两类 AI 能力：

- **标签建议 / 关系建议**：通过 `DeepSeek` 接口生成建议结果。
- **语义搜索**：通过 embedding 接口计算节点向量并进行相似度排序。

开发环境下依赖 Vite 代理：

- `/deepseek` → DeepSeek API
- `/relay` → OpenAI 兼容的 embedding 服务

说明：密钥通常通过本地 `.env.local` 和代理注入，不会直接打进前端代码。

## 使用方式

- 在左侧面板里切换星系、搜索、筛选、导入导出和重布局。
- 在画布中拖拽旋转、滚轮缩放、点击节点查看详情。
- 在右侧详情面板中编辑节点、添加关系、查看关联节点。
- 使用 `Ctrl/⌘ + K` 打开命令面板，快速创建节点、重新布局或切回示例数据。

## 项目结构

```text
src/
  ai/          # AI 接口封装（DeepSeek / embedding relay）
  components/  # 画布、面板、UI 组件
  data/        # 示例数据、数据库、可视化映射
  hooks/       # 持久化逻辑
  i18n/        # 文案
  store/       # Zustand 全局状态
  types/       # 领域类型定义
  utils/       # 搜索、筛选、布局、导入导出等工具
```

## 默认数据

首次启动时会生成一个默认星系，并写入内置示例图谱。示例数据适合直接体验以下能力：

- 多类型节点展示
- 节点间关系连线
- 标签筛选与时间筛选
- 搜索、语义搜索与 AI 建议

## 开发备注

- 该项目当前定位为本地优先的个人知识图谱工具。
- 数据主要保存在浏览器本地，不依赖后端数据库。
- 若要在生产环境继续使用 AI 能力，需要提供对应的代理或后端服务。
